# موتی‌گروپ | MovtiGroup Mirror
## مستندات آینه‌های نرم‌افزاری و داکر

این مخزن شامل کد منبع و محتوای وب‌سایت مستندات آینه‌های موتی‌گروپ است که با استفاده از **MkDocs** ساخته شده است.

### آدرس وب‌سایت
[https://doc.movtigroup.ir](https://doc.movtigroup.ir)

### محتویات مخزن
- **docs/**: فایل‌های منبع مستندات (Markdown) به زبان‌های مختلف.
- **mkdocs.yml**: تنظیمات اصلی وب‌سایت.
- **theme/**: شخصی‌سازی‌های قالب Material.
- **Dockerfile & docker-compose.yml**: تنظیمات مربوط به کانتینرایز کردن پروژه.

### نحوه مشارکت
برای افزودن مستندات جدید یا اصلاح ترجمه‌ها، می‌توانید یک Pull Request ارسال کنید. لطفاً قبل از ارسال، تغییرات خود را به صورت محلی با دستور `mkdocs serve` بررسی کنید.
