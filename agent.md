# Agent Instructions (AGENT.md)

This file provides instructions for AI assistants working on the MovtiGroup Mirror project.

## Project Scope
This project is a multi-language documentation site built with MkDocs and the Material theme. It documents Linux and Docker mirror services.

## Coding Standards
- Maintain multi-language support (fa, en, zh, ar, fr, de, tr).
- Ensure the footer always contains "MovtiGroup" and links to `doc.movtigroup.ir`.
- Use the standard URLs: `Linux.ththt.ir`, `install.ththt.ir`, and `docker.ththt.ir`.

## Verification
Always run `mkdocs build` after content changes to ensure there are no broken links or configuration errors.
