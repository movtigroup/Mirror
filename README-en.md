# MovtiGroup Mirror
## Software Mirror and Docker Documentation

This repository contains the source code and content for the MovtiGroup Mirror documentation website, built using **MkDocs**.

### Website URL
[https://doc.movtigroup.ir](https://doc.movtigroup.ir)

### Repository Contents
- **docs/**: Documentation source files (Markdown) in various languages.
- **mkdocs.yml**: Main website configuration.
- **theme/**: Material theme customizations.
- **Dockerfile & docker-compose.yml**: Containerization settings.

### How to Contribute
To add new documentation or fix translations, please submit a Pull Request. Check your changes locally using `mkdocs serve` before submitting.
