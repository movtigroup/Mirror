# Jules Agent Context (JULES.md)

Hello, I'm Jules. This file documents the context and decisions made during the initial development of this documentation site.

## Task Overview
- Goal: Create a documentation site for MovtiGroup Mirrors.
- Style: Inspired by `linuxmirrors.cn`.
- Tech Stack: MkDocs (Material theme), Docker, Nginx.

## Key Decisions
- **Multi-language**: Implemented using the `i18n` plugin. Persian (fa) is the default language.
- **Structure**: Home page features "One-click Script" commands for quick adoption.
- **Mirror URLs**: Hardcoded the requested `ththt.ir` subdomains as the primary access points.
