# مستندات لینوکس (Ubuntu/Debian)

در این بخش، نحوه استفاده از آینه‌های اختصاصی موتی‌گروپ برای سیستم‌عامل‌های مبتنی بر لینوکس (اوبونتو و دبیان) شرح داده شده است.

## استفاده از اسکریپت خودکار
سریع‌ترین راه برای تغییر مخازن، استفاده از سرویس `Linux.ththt.ir` است. این اسکریپت به طور خودکار نسخه سیستم‌عامل شما را تشخیص داده و بهترین آینه را جایگزین می‌کند.

```bash
bash <(curl -sSL https://Linux.ththt.ir)
```
