# 🐧 مستندات لینوکس (Ubuntu/Debian)

در این بخش، نحوه استفاده از آینه‌های اختصاصی موتی‌گروپ برای سیستم‌عامل‌های مبتنی بر لینوکس شرح داده شده است.

!!! info "نکته"
    آینه‌های موتی‌گروپ با هدف دور زدن محدودیت‌های شبکه و افزایش سرعت دانلود بسته‌ها طراحی شده‌اند.

## 🚀 استفاده از اسکریپت خودکار
سریع‌ترین راه برای تغییر مخازن، استفاده از اسکریپت‌های هوشمند ماست:

=== "🌐 دستور اصلی"
    ```bash
    bash <(curl -sSL https://Linux.ththt.ir)
    ```
=== "🔗 دستور کمکی"
    ```bash
    bash <(curl -sSL https://Linux.3cn.ir)
    ```
