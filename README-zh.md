# MovtiGroup 镜像
## 软件镜像和 Docker 工具的全面文档

此仓库包含 **MovtiGroup** 文档网站的源代码和内容。该网站旨在为开发人员提供详细的教程和一键安装脚本。

### 🚀 核心特性
- **支持 7 种语言**: 波斯语 (默认)、英语、中文、阿拉伯语、法语、德语和土耳其语。
- **一键脚本**:
    - Linux (Ubuntu/Debian): `Linux.ththt.ir`
    - Docker: `install.ththt.ir/docker.sh`
- **现代设计**: 使用 MkDocs Material 主题构建，支持深色/浅色模式切换。
- **生产就绪**: 包含 Docker 和 Nginx 配置。

### 🛠 技术栈
- **MkDocs**: 静态网站生成器。
- **Material for MkDocs**: 先进的响应式主题。
- **Docker & Docker Compose**: 用于容器化和简易运行。
- **Nginx**: 用于托管构建文件的生产服务器。

### 📦 使用 Docker 运行
在本地或服务器上运行网站：

```bash
docker-compose up -d --build
```
运行后，网站将在 **8080** 端口可用。
