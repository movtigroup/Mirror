### 📖 项目介绍
此仓库包含 **MovtiGroup** 文档网站的源代码。我们的目标是为开发人员提供快速且稳定的国际软件仓库访问。

### 🚀 智能安装脚本 (一键)
使用以下命令立即将您的系统连接到我们的镜像：

| 服务 | 主命令 (ththt.ir) | 辅助命令 (3cn.ir) |
| :--- | :--- | :--- |
| **Linux (Ubuntu/Debian)** | `bash <(curl -sSL https://Linux.ththt.ir)` | `bash <(curl -sSL https://Linux.3cn.ir)` |
| **安装 Docker 和镜像** | `bash <(curl -sSL https://install.ththt.ir/docker.sh)` | `bash <(curl -sSL https://install.3cn.ir/docker.sh)` |
| **仅设置 Docker 镜像** | `bash <(curl -sSL https://install.ththt.ir/mirror.sh)` | `bash <(curl -sSL https://install.3cn.ir/mirror.sh)` |

### 🛠 技术特性
- 🌍 **支持 7 种语言**: 波斯语、英语、中文、阿拉伯语、法语、德语和土耳其语。
- ⚡ **高性能**: 托管在强大的内容分发网络 (CDN) 上。
- 🐳 **Docker 就绪**: 使用 Docker Compose 轻松部署。
- 🎨 **Material UI**: 支持深色模式的现代简洁界面。

### 📦 快速开始
在本地运行网站：
```bash
docker-compose up -d --build
```
然后访问 `http://localhost:8080`。
